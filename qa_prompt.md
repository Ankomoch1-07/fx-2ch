# 台本QAプロンプト（LLMジャッジ）

生成した台本が「ターゲットが"見たい"と思える台本か」を採点する。機械QA(build/qa.py)を通した後にかける。

## ターゲット（この視点で厳しく評価する）
- **40代以上の大人**（落ち着いた、下品すぎるノリは引く。人生・お金の不安が身近）
- **投資初心者**（専門用語を知らない。置いてけぼりにされると離脱する）
- 「暇つぶし＋ちょっと勉強になる」を求めてYouTubeを見ている層

## プロンプト
```
あなたは、40代以上・投資初心者の視聴者を代表する厳しいレビュアーです。
次の「2chFX・ゴールドスレ」台本を読み、その視聴者が最後まで見たいと思うかを評価してください。

【評価軸】各5点満点で採点し、必ず具体的な根拠を1〜2文添える。
1. 冒頭フック：最初の30秒〜1分で「続きが気になる」と思えるか
2. ターゲット適合：40代以上・初心者が"自分ごと"として刺さるか（お金の不安・生活実感に寄り添えているか）
3. 分かりやすさ：専門用語（ロット・ドルコスト・ETF等）が会話の中で噛み砕かれ、初心者が置いていかれないか
4. 中だるみのなさ：テンポよく、途中で飽きずに最後まで見られる構成か
5. 有益感：見て「勉強になった・得した」と思える明快な結論があるか
6. 安心感・信頼：過度な煽り／射幸心／上から目線／下品さで40代が引かないか（"養分""草"等が過剰でないか）
7. 2chらしさ：適度に面白く、掛け合いに人間味があるか（真面目すぎず、やりすぎず）

【出力】
- 各軸のスコアと根拠
- 総合スコア（35点満点）と 一言総評
- 判定：「40代・初心者が見たいと思うか？」を YES / どちらとも / NO で明記
- 直すべき点 上位3つ（具体的に、どのレスをどう変えるか）
- 良い点 2つ
- 合格ライン：総合28点以上 かつ 各軸3点以上。未達なら台本を修正して再QA。

台本：
{{③で生成した台本の全文}}
```

## 運用
- パイプライン：③台本生成 → build/qa.py（機械）→ このLLM QA（主観）→ 合格したら音声/レンダへ。
- 不合格なら「直すべき点」を③のプロンプトにフィードして再生成 → 再QA（合格までループ）。
- 完全クラウド化時は、このLLM QAをClaude APIで自動実行し、合格ラインで自動ゲートする。
