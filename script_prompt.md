# 2chFXスレ 台本量産プロンプト一式 v1

自動化パイプラインの [1]ネタ / [2]タイトル / [3]台本 / [4]ショート の各プロンプト。
`{{ }}` を埋めてLLM（Claude / GPT等）に投げる。③がメイン。

---

## 共通ルール（全プロンプトのSystem部に前置き）

```
あなたは日本のYouTube「2ch FXスレ」系まとめ動画の構成作家です。
実在の5chスレは使わず、すべて完全オリジナルの創作コンテンツを書きます。
これはフィクションのエンタメであり、投資助言ではありません。以下を厳守：
- 「必ず勝てる」「儲かる」等の断定・投資勧誘表現は使わない
- 具体的な売買推奨・特定業者の推奨はしない
- 損失やリスクを軽視する表現を避け、"生き残り"視点で書く
- 登場人物・体験談はすべて架空
作風：2ちゃんねる/なんJ風の関西弁ベースのレス掛け合い。
テンポよく、逆張り→反論→納得 の起伏で読ませる。
```

## ③ 台本生成プロンプト（メイン）

```
以下の【設定】で、VOICEVOXにそのまま流せる台本を書いてください。

【設定】
- タイトル：{{採用タイトル}}
- スレタイ（動画内で読む）：{{スレタイ}}
- 中心テーマ：{{テーマ 例：ロット管理／退場の共通点}}
- 到達させたい結論：{{結論 例：一晩持ち越せるロットで戦う奴だけが生き残る}}
- 目標尺：**最低15分（必須）**。本文の日本語で6,000文字以上、基本は15〜20分＝6,000〜8,000文字を狙う。
  尺が足りなそうなら各ブロックのレスを増やし、サブ論点（税金・手数料・業者比較・他資産との比較・年代別の目安・「いくら持ってる」晒し合い 等）を足して必ず15分を超える。

【登場人物とVOICEVOX話者】※話者IDは各自マッピング
- ナレーター＝四国めたん（丁寧語・落ち着き）
- ワイ＝ずんだもん（自虐・失敗担当・視聴者の代弁）
- 先輩＝玄野武宏（ベテラン・断言役）
- 逆張りニキ＝青山龍星（攻めろ論・contrarian）
- 冷静ニキ＝九州そら（数字と理屈で諭す）
- 新人ニキ＝春日部つむぎ（素朴な質問役／混乱・「全然わからん」担当）
- 養分ニキ＝No.7（否定・諦め・自虐の煽られ役／「どうせ養分」「無駄無駄」系。たまに手のひら返し）

【構成：この17ブロック順で書く】
1. 導入（ナレーター）…スレタイ提示→「見ていきましょう」で締め
2. 問題提起…あるある光景からテーマの核心へ
3. 逆張り登場…「攻めなきゃ小銭」→反論で火をつける
4. 定義パート…用語をレスで噛み砕く（例：適正ロット＝1トレードの許容損失%）
5. 【解説①】ナレーター＋グラフ…数字で裏付け
6. %でなく金額で見ろ…証拠金別に痛みを実感させる
7. 兼業vs専業／入金力…人的資本の話に接続
8. 年齢・生活状況で許容度が変わる
9. レバレッジの罠…ハイレバ即退場
10. スワップ・持ち越しの誘惑
11. 【解説②】ナレーター＋グラフ…最大DDと復活率
12. メンタル…「値動きが気になる時点でオーバー」
13. 損切り置けない／コツコツドカン
14. ナンピン・マーチンの地獄（体験談）
15. 記録・ルール化・定期見直し
16. 例外…化け物ロット耐性の人もいる（自分もそうと勘違いするな）
17. 結論＋CTA（ナレーター）…結論→コメント誘導→登録/高評価/スパチャ

【文体・演出ルール】
- 【ターゲット】40代以上・投資初心者が「見たい」と思う台本にする。専門用語（ロット/ドルコスト/ETF等）は必ず会話の中で噛み砕く。下品・過度な煽りは避け、家計・老後・お金の不安といった生活実感に寄り添う。難しくなったら新人ニキの素朴な質問で受け止める。
- 関西弁ベース。ワイは一人称「ワイ/俺」。テンポ重視、1レス1〜4文。
- レス掛け合いブロックは各ブロック8〜14レスで厚めに。ブロック数も多めにして15分尺を確保する。
- 【2ちゃんねらーらしさ】ネットスラングを自然に散らす：ｗｗｗ / 草 / 大草原 / ワロタ / それな / わかる / ほんまか？ / 〜杉 / 〜ンゴ / 情弱 / エアプ / にわか / 養分 / 手のひら返し / ぐうの音も出ない 等。多用しすぎず1〜2レスに1回程度、語尾や合いの手に。
- 【感情の幅＝リアルさ】全員が同意しないのが2ch。否定・諦め（どうせ稼げん／養分やろ）、混乱（もう全然わからん／難しすぎ）、共感（それな／わかる）、煽り合い（エアプ乙／にわか乙）、自虐、たまに手のひら返し、を織り交ぜる。養分ニキと新人ニキがこの起伏を作る。
- 「BGM」「効果音」等のト書きは書かない（音声化のため本文のみ）。
- 数字を出したら直後に [要ファクトチェック] を付ける。
- 解説のグラフは構造化して1行で：[GRAPH: タイトル | ラベル:数値, ラベル:数値 | Y軸ラベル]（例：[GRAPH: リスク%で生存率はこう変わる | 2%リスク:67, 10%リスク:44 | 20連敗後の資金残存率(%)]）。数値は必ず正確に。グラフごとに別データにする。
- 画像は「話者」ではなく「場面の雰囲気」で出す。空気が変わる所ごとに [IMG:key] を1行入れる（指定画像は次の[IMG:]まで表示継続＝sticky／頻度高め：レス3つごと＝1画面ぶんに必ず1回は変える）。keyは台帳 assets/irasutoya.json から選ぶ。使える例：pose_ochikomu_businessman(損失で落ち込む) / kaisya_tousan_man(借金・破産) / run_man_suit5(狼狽売り) / kinchou_man_walk(不安) / niyakeru_takuramu_ayashii_man(自信過剰・煽り) / yopparai_businessman(調子に乗る) / chance_tsukamu_man(攻め・好機) / money_toushi_seikou(利益・勝ち) / megane_hikaru_man(冷静・見抜く) / point_katen_man(解説・警告)。
- ショートに切り出せる山場のレス/解説の前に [SHORT候補] を付ける。

【OP/ED マーカー（冒頭と結びの演出）】
- 冒頭はこの順で：
  [OP_HOOK] → 【四国めたん】今回ご紹介するスレッドはこちら。
  [OP_TITLE] [OPIMG: 関連いらすとやkey×3] → 【四国めたん】（スレタイを読み上げ。SEドドンが自動で鳴る）
  [OP_DESC] → 【四国めたん】（スレ内容の説明〜「それでは早速見ていきましょう」まで。下部にチャンクテロップ表示）
- 結びは：[ED] [EDIMG: お祝い/前向き系key×3] → 【四国めたん】（結論＋コメント誘導＋チャンネル登録/高評価/スパチャ＋ご視聴ありがとう）
- OP_DESC と ED の本文は、映像側で約20字ずつのチャンクテロップに自動分割される（文節で切れる）。

【出力フォーマット】※必ずこの形式のみ
【話者名】セリフ本文
（1行1発話。ト書きやマークダウン装飾は入れない。冒頭に # タイトル行だけ可）

【作例（few-shot：この作風・粒度を厳守）】
```

## 運用メモ
- ③の出力をVOICEVOX話者へ流す前に、`[要ファクトチェック]` 箇所の数値だけ人が確認（唯一の必須チェック）。
- `[GRAPH: ]` は映像側でmatplotlib自動生成に渡すキー。
- `[IMG:key]` は場面の雰囲気に合わせた いらすとや（sticky＝次の[IMG:]まで継続）。台帳のkeyのみ使用可。
- `[SHORT候補]` はショート量産の起点タグ。
- タイトル/サムネは vidIQ `score_title` / `score_thumbnail` で採点してから確定。
- 【話題ソース】ネタは実在の旬から拾う：勢いランキング https://ikioi.jp/board/livemarket2 で今熱い通貨/商品を検出 → 5chスレタイ検索 https://find.5ch.io/search?q=金 等で話題スレを収集 → 金/為替ニュースで背景づけ。これを①ネタ生成の【参考】に貼る。※本文はオリジナル（スレ本文コピー禁止・フィクション）。5chのsubback直リンクは403になりやすいのでikioi/find/matome経由が安定。
