BGM（バックミュージック）をここに置きます。ロイヤリティフリー/商用可の音源を使ってください。

■ 使い方（2ファイル）
- OPとEDで流す曲 … ファイル名に op / ed / open のいずれかを含める（例: op_ed.mp3）
- 本編(中間)で流す曲 … ファイル名に main / honpen のいずれかを含める（例: main.mp3）
  ※命名しない場合は、ファイル名の昇順で1本目=OP/ED、2本目=本編 に自動割当。

対応形式: .mp3 / .wav 。tts.py が自動で timeline に載せ、Board.tsx が
OP=[冒頭〜本編開始) / 本編=[本編〜ED開始) / ED=[ED〜最後] で切り替えて
ループ＋小音量＋両端フェードで再生します（音量は Board.tsx の BGM_OP/BGM_MAIN）。
